# Sifr Language Support for VSCode

إضافة رسمية للغة **صِفر (Sifr)** — لغة البرمجة العربية الأصيلة.

## المزايا

- ✨ تلوين الكلمات المفتاحية العربية: `كنز`، `سرّ`، `مهمّة`، `إن`، `جوال`، `كرر`، `طابق`، `بنية`، `وحدة` …
- 🎨 تلوين الدوال المدمجة: `أرني`، `خريطة`، `تصفية`، `شبكة_عصبية`، `قاموس`، `تيار` …
- 💬 دعم التعليقات `//` و `/* */`
- 📝 إغلاق تلقائي للأقواس وعلامات التنصيص
- 🔤 اقتراح المسافة البادئة بعد `:` وتقليلها عند `انتهى`
- 📁 الامتدادات المدعومة: `.sifr`, `.صفر`, `.arb`

## التثبيت اليدوي

1. انسخ مجلد `vscode-sifr` إلى:
   - **macOS / Linux:** `~/.vscode/extensions/sifr-lang-1.0.0`
   - **Windows:** `%USERPROFILE%\.vscode\extensions\sifr-lang-1.0.0`
2. أعد تشغيل VSCode.
3. افتح ملفاً بامتداد `.sifr` — سيُفعَّل التلوين تلقائياً.

## بناء حزمة `.vsix` (اختياري)

```bash
npm install -g @vscode/vsce
cd vscode-sifr
vsce package
code --install-extension sifr-lang-1.0.0.vsix
```

## مثال

```sifr
// 🌙 برنامج صِفر بسيط
كنز اسم = "محمد"
مهمّة حيّ(شخص) :
    أرني("مرحباً يا " + شخص)
انتهى
حيّ(اسم)
```
