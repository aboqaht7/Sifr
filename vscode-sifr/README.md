# Sifr — لغة صِفر العربية | VSCode Extension

إضافة Visual Studio Code الرسميّة لـ **صِفر (Sifr)** — أوّل لغة برمجة عربية أصيلة بصياغة فريدة من نوعها.

## ✨ ما تقدّمه الإضافة

- 🎨 **تلوين الصياغة** الكامل لجميع الكلمات المفتاحيّة العربيّة
- ⚡ **324 إكمالاً ذكياً (Snippets)** تشمل:
  - الكلمات المفتاحيّة (`كنز` ، `مهمّة` ، `إن` ، `جوال` ، `كرر` ، `طابق` ، `حاول` ، `بنية` ، `وحدة` ، …)
  - جميع الـ **311 دالة مدمجة** مع توقيعات معاملاتها
- 📁 **دعم الامتدادات** `.sifr` و `.صفر` و `.arb`
- 🌍 **دعم RTL** الكامل
- 📘 **مرجع شامل** للّغة في `docs/REFERENCE.md`

## 🚀 الاستخدام

افتح أيّ ملف بامتداد `.sifr` وسيتفعّل التلوين والإكمال تلقائياً. اكتب أيّ كلمة مفتاحيّة (مثل `كنز` أو `مهمّة`) واضغط `Tab` ليتمّ توليد القالب.

### مثال سريع

```sifr
كنز الاسم = "محمد"
سرّ السنة = 1446

مهمّة سلّم(اسم) :
  أرني("السلام عليك يا " + اسم)
انتهى

سلّم(الاسم)
```

## 📚 المرجع الكامل

كلّ كلمات اللغة ودوالها موثّقة في **[docs/REFERENCE.md](./docs/REFERENCE.md)** — يحتوي على:
- 27 كلمة مفتاحيّة
- 311 دالة مدمجة موزّعة على 19 فئة
- 7 هياكل بيانات متقدّمة (HashMap ، Set ، Queue ، Stack ، Heap ، LinkedList ، BST)
- مكتبة Streams ، خوارزميات ، ذكاء اصطناعي ، DOM API ، أعداد كبيرة …

## 🛠️ التثبيت

```bash
# من VSIX:
code --install-extension sifr-lang-1.1.0.vsix

# أو نسخ المجلّد إلى:
~/.vscode/extensions/sifr-lang-1.1.0/
```
